CSS Variablen

CSS Variablen heißen Custom Properties.
Für die Bezeichner gilt: sprechend und nicht wertorientiert.
Custom Properties müssen innerhalb einer CSS Regel vereinbart werden.
Der scope ist der jeweilige Selektor.
Einen global scope erreichen wir mit einem der Selektoren html, body oder :root.

Mehr zu Custom Properties findet man z.B. bei CSS Tricks

Die Elementbox

Die Elementbox ist gut in den developer tools des browser zu erkennen.

Bestandteile der Elementbox

  • content: width und height
  • padding
  • border
  • margin

Die properties haben jeweils vier Kanten, die alle einzeln behandelt werden können.

Mit dem margin kann auch eine Platzierung vorgenommen werden

Größenangaben für Elemente

Zusätzlich zu den Längenmaßeinheiten können Schlüsselwörter vergeben werden.

Die width kann intuitiv vergeben werden.

Für die height gelten besondere Regeln.

Berechnung von width

Die Berechnung der Breite und Höhe funktionieren gleich. Die Höhe sollte außerhalb von Grid- und Flexitems unangetastet bleiben, deshalb wird hier nur die Breite erwähnt.

Berechnungsgrundlage ist das box-sizing.

Das Display Property

Elemente haben für das display property einen der drei folgenden Werte:

  • block
  • inline
  • inline-block

Für Block Elemente gelten alle Regeln, die vorab zu den Box Properties aufgelistet sind.

Für Inline Elemente gelten Sonderregeln. Inline Elemente haben per default die Größe ihres Inhaltes. D.h. man kann ihnen keine Größe zuweisen. Margins können nur horizontal zugewiesen werden. Wenn genug Platz ist, dann können mehrere Inline Elemente nebeneinander platziert werden.

Bilder

Bilder sind ersetzte inline Elemente. Sie haben eine inherente Größe.

Bilder sollten grundsätzlich auf die Größe des Elternelementes begrenzt werden.
